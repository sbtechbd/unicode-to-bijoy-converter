A WordPress plugin to convert Bangla text between Unicode and Bijoy formats, with voice typing support and a modern Tailwind CSS interface.

== Description ==

**Unicode to Bijoy Converter** is a lightweight plugin that allows users to easily convert Bangla text between Unicode and Bijoy formats. It also features Bangla voice typing support, making text input more accessible. The plugin uses Tailwind CSS for a modern and responsive user interface.

### Features
- **Unicode to Bijoy conversion**: Easily convert Unicode Bangla text to Bijoy format.
- **Bijoy to Unicode conversion**: Convert Bijoy text back to Unicode format.
- **Voice typing support**: Use your voice to type Bangla text with the built-in `webkitSpeechRecognition` API.
- **Clean, responsive UI**: Tailwind CSS ensures the interface looks great on any device.

### Usage
- After activating the plugin, use the `[unicode_to_bijoy_converter]` shortcode to embed the converter on any page or post.

### Why Use This Plugin?
If you're a Bangla content creator or publisher, this plugin simplifies the process of working with Unicode and Bijoy formats. Whether you're typing, pasting, or converting text, this tool makes it fast and efficient.

== Installation ==

1. Download the plugin from the WordPress plugin repository or upload the `.zip` file via the WordPress admin panel.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Add the shortcode `[unicode_to_bijoy_converter]` to any page or post to display the converter.

== Frequently Asked Questions ==

= Does this plugin work with all themes? =  
Yes, the plugin is designed to work with any WordPress theme.

= How do I enable voice typing? =  
Simply click the "Click Here to Voice Typing" button on the converter interface, and start speaking in Bangla.

= Is the plugin mobile-friendly? =  
Yes, the interface is built with Tailwind CSS and is fully responsive.

= What are the browser requirements for voice typing? =  
Voice typing requires a browser that supports the `webkitSpeechRecognition` API, such as Google Chrome.
